Title: Hangman Game
Goal:
Create a simple text-based Hangman game where the player guesses a hidden word by entering one letter at a time.
Description:
The computer randomly selects a word from a predefined list of words. The player tries to guess the word by entering letters. If the guessed letter is correct, it is displayed in the correct position. If the guess is wrong, the player loses one chance.
The player gets a maximum of 6 incorrect guesses. The game ends when:
The player guesses the complete word (Win)
The player reaches 6 wrong guesses (Game Over)
Features:
Uses 5 predefined words
Random word selection
Letter-by-letter guessing
Tracks correct and incorrect guesses
Displays remaining attempts
Console-based input/output
Concepts Used:
Random module
Selects a random word from the word list.
